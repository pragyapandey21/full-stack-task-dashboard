# Full Stack Task Dashboard 🚀

A full-stack task management dashboard built using HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, JWT Authentication, Render, and Vercel.

## 🌐 Live Demo
https://full-stack-task-dashboard.vercel.app

---

## ✨ Features

- User Registration & Login Authentication
- JWT-based Authentication
- Add / Delete / Complete Tasks
- Task Analytics Dashboard
- Subject-based Task Filtering
- Responsive UI
- MongoDB Database Integration
- Full-stack Deployment

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Deployment
- Vercel (Frontend)
- Render (Backend)
